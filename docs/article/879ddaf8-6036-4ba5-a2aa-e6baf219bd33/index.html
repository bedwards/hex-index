  • List of mergers and acquisitions by Alphabet

    This list provides the raw data showing that Google's spending on three security firms (Mandiant, Siemplify, Wiz) nearly equals its spending on all other disclosed acquisitions combined, proving the article's thesis that platform giants are pivoting to security.

  • Zero trust architecture

    The article argues that platforms must become security vendors; this concept explains the specific technical paradigm shift from 'trust but verify' to 'never trust, always verify' that forces cloud providers like Google and Microsoft to embed security directly into their infrastructure rather than selling it as an add-on.

  • Cloud computing security

    This cloud computing concept defines the boundary between provider and customer security duties, illuminating why the article claims platforms are 'becoming' security companies: as enterprises move critical workloads to the cloud, the platform vendor is forced to absorb more security liability to prevent customers from leaving.
