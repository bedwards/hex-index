How To Actually Be Anonymous

The Hated One · The Hated One ·Sep 27, 2020 · 10 min read
Commentary by Brian Edwards
Original video →

The Hated One makes a provocative claim that upends the common assumption that anonymity is a tool reserved for criminals: "Internet anonymity isn't just for the most secretive and shady people; anonymity should be the default for everyone." In an era where data brokers and governments treat every click as a public record, the author argues that the only way to reclaim privacy is to adopt a specialized operating system called Whonix. This is not a casual browser extension or a simple setting change; it is a fundamental architectural shift that forces all internet traffic through a virtual machine, creating a barrier that standard operating systems simply cannot provide.

The Architecture of Isolation

The core of The Hated One's argument rests on the concept of "security by compartmentalization," a method described by mainstream security researchers as the gold standard for system defense. Unlike a standard browser that runs directly on your computer, Whonix runs inside a virtual machine, effectively creating a digital sandbox. The author explains that "Linux runs in isolation from your main system so that even if your VM gets infected with malware, Whonix would usually not allow it to escape into your host operating system."

This isolation is critical because the internet was never built with security in mind. As The Hated One notes, "All connections made on the internet reveal true physical locations and everything is being recorded by advertisers, data brokers, and governments." By routing traffic through two separate virtual machines—a "Gateway" that handles the connection and a "Workstation" where the user operates—the system ensures that even if the user visits a malicious website, their actual IP address remains hidden. The author emphasizes that this setup "eliminates the possibility of a potential leak of your IP address," a vulnerability that plagues standard Tor browser usage on Windows or macOS.

Critics might note that running a full virtual machine requires significant hardware resources, potentially making this solution inaccessible for users with older computers. However, The Hated One counters this by suggesting that users can dedicate specific amounts of RAM and CPU power, noting that the system "will only take it when in use."

"Tor makes it so that every user looks the same and nobody's true location is revealed to the open internet. Whonix replicates this process but with some additional security features."

Beyond the Browser: A Daily Driver

The most distinctive part of this coverage is the push to move beyond using anonymity tools only when necessary. The Hated One argues that Whonix is designed to be a "daily driver," stating that "running it as your daily driver isn't really that much different than using your traditional operating system." The software comes pre-loaded with essential tools, including a password manager and cryptocurrency wallets, allowing users to integrate privacy into their mundane tasks.

The author highlights the importance of cryptocurrency privacy, specifically contrasting Bitcoin with Monero. While Bitcoin transactions are publicly broadcast and traceable, Monero offers an "opaque blockchain" where transactions are "only visible to you and your recipients." The Hated One writes, "Monero is the true cash of cryptocurrency," and suggests that routing these transactions through Tor while using Whonix provides a level of financial privacy that standard banking or even standard crypto usage cannot match. This section is particularly compelling because it moves the conversation from theoretical privacy to practical, everyday financial security.

Furthermore, the author addresses the fear that such high-security tools are difficult to use. The installation process is described as straightforward: install VirtualBox, import the Whonix appliance, and launch the two virtual machines. The Hated One reassures readers that "it is really simple," demystifying a process that often intimidates the average user. The inclusion of a "live mode," where files are erased immediately after powering off, adds an extra layer of security against persistent malware, though the author notes this comes at the cost of not saving files between sessions.

The Culture of Anonymity

Ultimately, The Hated One frames anonymity not as a shield for the guilty, but as a necessary defense for the innocent. The argument extends to journalists, activists, and researchers who need to protect their sources and their own safety. "The uses are limitless," the author asserts, suggesting that compartmentalizing internet activities into different virtual machines can protect everything from social media accounts to general browsing habits.

The commentary concludes with a vision of a more secure internet ecosystem. The Hated One posits that "the more people become anonymous, the better protection everyone gets," implying that anonymity is a collective good rather than an individual luxury. While the tutorial focuses on technical implementation, the underlying message is a call to action for digital self-defense. The author admits that running Whonix on a security-focused operating system like Qubes OS offers an even higher level of protection, though that is beyond the scope of this specific guide.

"Internet anonymity isn't just for the most secretive and shady people; anonymity should be the default for everyone."

Bottom Line

The Hated One's strongest argument is the reframing of anonymity as a default necessity rather than a niche tool for the paranoid, backed by a robust technical explanation of why virtualization beats standard browser settings. The piece's biggest vulnerability lies in the assumption that average users have the hardware capacity and technical patience to maintain a virtualized environment for daily tasks. Readers should watch for the balance between this heightened security and the inevitable friction it introduces into everyday digital life.
