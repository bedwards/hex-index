# How do I subscribe?

To subscribe to this newsletter, go to [historycanthide.substack.com/subscribe.](http://historycanthide.substack.com/subscribe)

There are three subscription types:

1.  **Monthly** - $5 a month
    
2.  **Annual** - $50 a year
    
3.  **True Investigator** - You select the amount (>$150 a year)
    

The website default selects the annual plan, but if you want to contribute more, select True Investigator and input an amount in the submission box (see below).
